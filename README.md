# Blog app

This is a blog application built with Ruby on Rails. 
- This app has the features of uploading images as well as posting comments.

![screenshot](./public/screenshot-blog.png)

## Tools/Languages Used

* Ruby on rails
* Heroku
* Sass
* Unsplash source API
